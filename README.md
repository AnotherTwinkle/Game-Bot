# Game-Bot

This Discord bot takes in a Steam game's name and returns its link on the Steam Store based on Valve's search suggest feature.

## Dependencies

Discord.py:
```
python3 -m pip install -U discord.py
```

asyncio:
```
pip install asyncio
```

**Make sure you are running Python 3.4.2+!**
